As part of an assignment of a coding course I am taking at Mizzou, I made a red heart that says "I love you, dawg". I used this opportunity to send this funny file to a friend of mine that also uses python.

here is a snippet of the code for this project in Python. 
   
            import turtle
      pen=turtle.Turtle()

      def curve():
          for i in range (200):

              pen.right(1)
                pen.forward(1)

      def heart():
          pen.fillcolor('red')
          pen.begin_fill()

          pen.left(140)
          pen.forward(113)
                  
          curve()
          pen.left(120)

          curve()
          pen.forward(112)
          pen.end_fill()

      def txt():
          pen.up()
          pen.setpos(-68,95)
          pen.down()
          pen.color('pink')
          pen.write("Love you, Dawg", font=("verdana", 14, "bold"))
    
      heart()

      txt()

      pen.ht()
